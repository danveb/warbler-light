<h1>Setup</h1>

Clone repo

```sh
$ git clone git@github.com:danveb/warbler-lite.git
```

Head to client on project directory where package.json/yarn.lock are located

```sh
$ cd/warbler-lite/client
```

Install required dependencies 

```sh
$ yarn install
```

Start the server 

```sh
$ yarn start
```
